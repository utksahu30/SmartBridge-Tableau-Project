Decoding the DNA of Real Estate

Merging data science with visual storytelling to uncover the true drivers of property value in a shifting market.

The Vision

This project translates complex housing variables into clear, interactive insight for buyers, sellers, and analysts. The objective is to make market intelligence understandable and actionable.

Tech Stack

Tableau Public for data storytelling, Python workflows for preparation, and semantic HTML/CSS for presentation.

Project Objectives

Identify price-driving features, visualize neighborhood-level behavior, and support exploration through interactive dashboards.
